# Webpack 4 Template
Webpack 4 with AVA Template

## To Install
npm install

## Initialize Lint
npm run lint-init

## Run Lint
npm run lint

## Run AVA Tests
npm test

## Run localhost
npm run watch

## Create developer build - no caching or uglifyjs
npm run build-dev

## Create production build -caching and uglifyjs
npm run build-release
